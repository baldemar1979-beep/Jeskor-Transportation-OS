# Fleet KPI Scorecard

## Purpose

This document establishes key fleet performance indicators used to measure tractor reliability, maintenance effectiveness, and operational performance.

## Uptime Metrics

### Tractor Availability

Target:

* Greater than 98%

### Downtime Hours

Track monthly.

## Maintenance Metrics

### PM Compliance

Target:

* 100%

### Repeat Repairs

Track recurring failures and repair trends.

## Breakdown Metrics

### Breakdown Frequency

Track:

* Monthly events
* Events per tractor

### Roadside Events

Track monthly.

## Cost Metrics

### Maintenance Cost Per Tractor

Monitor monthly.

### Roadside Repair Cost

Monitor monthly.

## Vendor Metrics

### Vendor Response Time

Track average response time.

### Repair Completion Time

Track average completion time.

## Review Frequency

Review monthly.

Review quarterly with leadership.

## Objective

Provide leadership with visibility into fleet performance, reliability, compliance, and maintenance effectiveness.
