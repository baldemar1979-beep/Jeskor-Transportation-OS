# Amazon RACI Enforcement Model

## Source

AFP US Non-Compliance Enforcement Matrix - RACI Model

## Purpose

This document summarizes Amazon's enforcement ownership model and identifies how accountability is assigned during investigations, corrective actions, enforcement reviews, and closure activities.

The objective is to incorporate similar accountability principles into Jeskor Transportation's operating system.

## What Is RACI?

RACI is a responsibility assignment framework.

- Responsible = Performs the work
- Accountable = Owns the outcome
- Consulted = Provides input
- Informed = Receives updates

Amazon uses this framework to ensure enforcement actions are properly managed and documented.

## Key Observation

Amazon assigns ownership to every step of the enforcement process.

This includes:

- Identifying issues
- Investigating issues
- Developing corrective actions
- Monitoring completion
- Verifying effectiveness
- Closing actions

## Enforcement Lifecycle

### Step 1

Issue Identification

Examples:

- Safety event
- Compliance failure
- Operational deficiency

### Step 2

Investigation

Determine:

- Root cause
- Contributing factors
- Risk level

### Step 3

Corrective Action Plan

Develop:

- Action items
- Owners
- Due dates

### Step 4

Monitoring

Track:

- Completion status
- Progress updates
- Effectiveness

### Step 5

Closure Verification

Verify:

- Corrective actions completed
- Issue resolved
- Risk reduced

## Lessons Learned

### Lesson 1

Every issue must have an owner.

### Lesson 2

Corrective actions require due dates.

### Lesson 3

Completion should be verified.

### Lesson 4

Documentation is critical.

### Lesson 5

Management accountability is a key component of enforcement.

## Jeskor Transportation Applications

For future investigations, audits, CAPs, and safety reviews, track:

- Issue
- Owner
- Due date
- Status
- Completion date
- Verification date

## Future Dashboard Recommendations

Monitor:

- Open actions

## Conclusion

Amazon's enforcement process relies on clear ownership, accountability, documentation, and verification. Jeskor Transportation should adopt similar practices to strengthen compliance, safety, and operational performance.
