# Q2 2025 Enforcement Matrix Insights

## Source

AFP US Non-Compliance Enforcement Matrix (Version 4.2.25)

## Summary

This document captures key lessons learned from Amazon's enforcement framework and identifies actions Jeskor Transportation can take to proactively manage enforcement risk.

## Key Observations

Amazon primarily evaluates:

- Severity
- Frequency
- Repeat behavior
- Failure to correct

Most enforcement actions occur after patterns emerge rather than isolated incidents.

## Escalation Path

1. Courtesy Notification
2. Corrective Action Plan (CAP)
3. Business Operations Review (BOR)
4. Offboarding Risk

## High-Risk Categories

- Driver qualification deficiencies
- Drug and alcohol violations
- Insurance deficiencies
- Regulatory non-compliance
- Service failures
- Fraud
- Falsified records

## Lessons Learned

### Track Trends

Recurring issues create more risk than isolated events.

### Document Corrective Actions

Undocumented corrective actions may be viewed as incomplete.

### Escalate Early

Small issues become larger risks when left unresolved.

### Assign Ownership

Every corrective action should have an owner and due date.

### Verify Effectiveness

Corrective actions should be validated for effectiveness, not simply marked complete.

## Jeskor Applications

Future management systems should track:

- Violation category
- Violation count
- Corrective action status
- Escalation level
- Repeat offender trends

## Future Dashboard Recommendations

Monitor:

- Courtesy Notifications
- CAP Exposure
- Open Corrective Actions
- Repeat Violations
- Compliance Trends

## Conclusion

Amazon's enforcement framework focuses on recurring patterns and management effectiveness. Jeskor Transportation should focus on proactive monitoring, corrective action management, and trend analysis to reduce enforcement exposure and maintain AFP program eligibility.
