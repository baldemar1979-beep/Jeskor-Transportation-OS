# Missed Load Response Plan

## Purpose

This document establishes the response process for missed loads and service failures.

---

# Objectives

- Recover service quickly
- Minimize customer impact
- Identify root causes
- Prevent recurrence

---

# Immediate Actions

Determine:

- Cause of missed load
- Current load status
- Recovery options

---

# Escalation

Notify:

- Operations
- Dispatch leadership
- Customer (when required)

---

# Root Cause Categories

- Driver availability
- Equipment failure
- Dispatch error
- Customer issue
- Weather
- Traffic

---

# Recovery Actions

Examples:

- Reassign driver
- Reassign equipment
- Alternate route
- Customer coordination

---

# Documentation

Record:

- Incident summary
- Root cause
- Recovery actions
- Lessons learned

---

# Continuous Improvement

Review missed loads monthly and identify recurring trends.
