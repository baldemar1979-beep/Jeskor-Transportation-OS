# Missed Block Response Plan

## Purpose

This document establishes procedures for responding to missed blocks and at-risk blocks.

---

# Objectives

- Protect service
- Minimize customer impact
- Recover operations quickly
- Prevent recurrence

---

# Missed Block Triggers

Examples:

- Driver no-show
- Equipment failure
- Driver illness
- Scheduling error

---

# Immediate Response

Determine:

- Cause
- Available drivers
- Available equipment
- Recovery options

---

# Escalation

Notify:

- Dispatch leadership
- Operations leadership
- Safety (if applicable)

---

# Recovery Options

Examples:

- On-call driver
- Schedule adjustment
- Equipment reassignment
- Alternate coverage

---

# Root Cause Analysis

Determine:

- Staffing issue
- Equipment issue
- Process failure
- Communication failure

---

# Documentation

Record:

- Block information
- Cause
- Recovery actions
- Final outcome

---

# Continuous Improvement

Review missed blocks monthly and identify recurring causes.
