# Recruiting Sourcing

## Purpose

This repository contains candidate sourcing strategies and recruiting channels used by Jeskor Transportation.

## Contents

- Indeed recruiting strategies
- Referral programs
- Military recruiting initiatives
- CDL school partnerships
- Candidate pipeline development

## Objective

Create a repeatable system for attracting qualified candidates while maintaining recruiting efficiency and hiring quality.

## Success Metrics

- Applicant volume
- Qualified applicant percentage
- Cost per hire
- Time to fill
- Candidate quality
