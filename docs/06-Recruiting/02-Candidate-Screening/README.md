# Candidate Screening

## Purpose

This repository contains the screening standards used to evaluate driver candidates before hiring.

## Contents

- Driver verification process
- MVR review guide
- Employment verification process
- Reference check process
- Functional driver assessment
- Candidate scoring framework

## Objective

Identify safe, qualified, coachable, and technology-friendly drivers before they enter the hiring pipeline.

## Screening Principles

- Safety first
- Verify before hire
- Use consistent scoring
- Document all decisions
- Avoid subjective or appearance-based judgments
