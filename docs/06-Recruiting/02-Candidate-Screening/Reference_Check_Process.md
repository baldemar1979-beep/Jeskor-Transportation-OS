# Reference Check Process

## Purpose

This document establishes the standard reference check process for driver candidates.

The objective is to verify employment history, evaluate candidate reliability, and identify potential safety, attendance, or performance concerns before hire.

---

# Reference Requirements

Minimum:

- Two professional references

Preferred:

- Previous supervisor
- Previous transportation manager
- Fleet manager
- Operations manager

Personal references should only be used when professional references are unavailable.

---

# Verification Requirements

Confirm:

- Employment dates
- Position held
- Equipment operated
- Reason for separation
- Eligibility for rehire

---

# Standard Questions

## Employment Verification

Was the candidate employed with your company?

What position did they hold?

How long were they employed?

---

## Performance

How would you describe the candidate's work performance?

Did they reliably complete assigned work?

Were they dependable?

---

## Attendance

How was the candidate's attendance?

Were there any recurring attendance concerns?

---

## Safety

Did the candidate have any preventable accidents?

Were there any safety violations?

Would you consider the candidate a safety-focused employee?

---

## Professionalism

How did the candidate interact with supervisors?

How did the candidate interact with coworkers?

Did the candidate respond well to coaching?

---

## Rehire Question

Would you hire this candidate again?

Why or why not?

---

# Red Flags

Additional review required for:

- Refusal to discuss candidate
- Negative safety feedback
- Attendance concerns
- Multiple policy violations
- Performance concerns
- Inconsistent employment history

---

# Scoring

## Strong Reference

10 Points

## Positive Reference

8 Points

## Neutral Reference

5 Points

## Concerning Reference

2 Points

## Negative Reference

0 Points

---

# Documentation

Record:

- Reference name
- Company
- Position
- Contact information
- Date contacted
- Interview notes

Retain records in candidate file.

---

# Hiring Guidance

References should support information provided during:

- Application review
- Interview process
- Verification process

Inconsistencies should be investigated before hire approval.
