# Interview Process

## Purpose

This repository contains structured interview tools for evaluating candidates consistently.

## Contents

- Driver interview questions
- Safety interview guide
- Technology assessment
- Behavioral interview questions
- Candidate scoring guide

## Objective

Create a repeatable interview process that identifies candidates who align with Jeskor Transportation's safety standards, communication expectations, and AFP operating requirements.

## Interview Principles

- Ask consistent questions
- Evaluate safety mindset
- Evaluate communication
- Evaluate technology comfort
- Document candidate responses
