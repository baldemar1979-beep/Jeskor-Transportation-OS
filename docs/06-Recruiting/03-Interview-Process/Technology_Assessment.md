# Technology Assessment

## Purpose

This assessment evaluates a candidate's ability to operate within a technology-driven AFP environment.

---

# Assessment Areas

## Mobile Applications

Evaluate:

- Smartphone usage
- App navigation
- Photo uploads

---

## ELD Experience

Evaluate:

- Previous ELD usage
- Comfort level
- Training requirements

---

## Driver Facing Cameras

Discuss:

- Netradyne
- Safety technology
- Coaching programs

Evaluate reaction.

---

## Navigation Systems

Evaluate:

- GPS usage
- Route planning
- Location sharing

---

## Technology Exercise

Ask candidate to:

- Open a mobile application
- Navigate menus
- Upload a photo

Observe:

- Confidence
- Adaptability
- Learning ability

---

# Scoring

1-10
