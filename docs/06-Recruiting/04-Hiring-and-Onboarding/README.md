# Hiring and Onboarding

## Purpose

This repository contains the hiring and onboarding process for new employees.

## Contents

- New hire checklist
- Driver onboarding process
- Day one orientation
- Compliance onboarding
- Training requirements
- Driver activation process

## Objective

Ensure every new hire is properly documented, trained, activated, and prepared before operating under Jeskor Transportation.

## Onboarding Principles

- Complete documentation before dispatch
- Confirm training completion
- Verify compliance requirements
- Confirm technology access
- Set clear safety expectations
