# Corrective Action Framework

## Purpose

This document establishes a structured corrective action process for safety-related performance issues.

---

# Philosophy

Corrective action should:

- Improve behavior
- Reduce risk
- Support accountability
- Promote consistency

---

# Level 1

Coaching

Examples:

- Minor violations
- First-time events

---

# Level 2

Written Warning

Examples:

- Repeat violations
- Significant safety concerns

---

# Level 3

Final Warning

Examples:

- Serious violations
- Failure to improve

---

# Level 4

Termination Review

Examples:

- Gross negligence
- Severe safety violations
- Repeated non-compliance

---

# Review Factors

Consider:

- Severity
- Frequency
- Driver history
- Corrective actions completed

---

# Documentation

Maintain:

- Event Summary
- Coaching Records
- Warnings Issued
- Follow-Up Actions

---

# Goal

Create fair, consistent, and defensible safety-related corrective actions.
