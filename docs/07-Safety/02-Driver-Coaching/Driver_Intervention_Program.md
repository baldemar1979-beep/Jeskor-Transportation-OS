# Driver Intervention Program

## Purpose

This document establishes the process for intervening with drivers demonstrating elevated safety risk.

## Objective

Identify high-risk drivers early and implement corrective actions before accidents occur.

---

# Intervention Triggers

Examples:

- Severe fatigue event
- Repeat fatigue events
- Repeat distraction events
- Elevated risk score
- Preventable accident
- Multiple coaching events

---

# Intervention Levels

## Level 1

Informal Coaching

Examples:

- Single severe event
- Minor repeat behavior

Actions:

- Coaching discussion
- Education
- Monitoring

---

## Level 2

Formal Intervention

Examples:

- Multiple severe events
- Elevated risk score

Actions:

- Supervisor meeting
- Development plan
- Increased monitoring

---

## Level 3

Management Review

Examples:

- Repeat intervention failure
- Serious safety concerns

Actions:

- Safety review board
- Corrective action review
- Return-to-duty assessment

---

# Documentation

Maintain:

- Trigger
- Intervention level
- Corrective actions
- Follow-up dates
- Outcomes

---

# Program Goal

Reduce risk through early intervention and driver development.
