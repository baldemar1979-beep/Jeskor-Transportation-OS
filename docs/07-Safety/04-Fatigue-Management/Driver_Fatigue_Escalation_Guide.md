# Driver Fatigue Escalation Guide

## Purpose

This guide establishes escalation requirements for fatigue-related events.

---

# Level 1

Single Low-Risk Event

Response:

- Coaching
- Education
- Monitoring

---

# Level 2

Repeated Fatigue Events

Response:

- Supervisor Review
- Driver Discussion
- Increased Monitoring

---

# Level 3

Severe Fatigue Event

Examples:

- Eyes Closed
- Microsleep
- Severe Drowsiness

Response:

- Immediate Review
- Operations Notification
- Potential Driver Removal

---

# Level 4

Repeat Severe Fatigue Events

Response:

- Management Review
- Corrective Action Review
- Return-to-Duty Assessment

---

# Documentation

Track:

- Event Type
- Severity
- Escalation Level
- Actions Taken

---

# Program Goal

Reduce fatigue-related risk while ensuring drivers receive appropriate support and intervention.
